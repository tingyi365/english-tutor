# 🗣️ AI 英語口說老師 · Speak English

像真人英文老師一樣陪你練英文的**純前端**互動學習網站。核心是**語音聽說即時糾正**：開口說，老師立刻逐字標出對與錯，並給具體的中文發音建議。

> 免註冊、免後端、免任何 API 金鑰即可完整運作。建議使用 **Chrome**（桌面版或 Android）並允許麥克風權限。

## ✨ 功能

| 模式 | 說明 |
| --- | --- |
| 🎤 跟讀糾音 | Web Speech 聽你發音，逐詞比對目標句、高亮對/錯/漏字，給發音評分與糾正建議 |
| ✍️ 聽寫練習 | 只播聲音，把句子打出來，訓練聽力與拼寫，逐字對答案 |
| 💬 情境對話 | 真實情境一來一往（點餐、認識朋友、問路），開口回應並評分 |
| 🃏 單字卡 | 翻卡記單字，一點即聽單字與例句發音 |
| 📝 文法填空 | 單選題練文法，附中文解析 |

- **介面**：繁體中文；**學習內容**：英文。
- **響應式**：手機 / 桌面皆可用。
- **技術**：Vanilla JS（ES Modules）+ Web Speech API（SpeechRecognition / SpeechSynthesis），無建置步驟。

## 🧠 發音糾正原理

`assets/js/scoring.js` 以編輯距離做「目標句 ↔ 辨識結果」的逐詞動態規劃對齊，判定每個字為
**正確 / 近音 / 唸錯 / 漏字 / 多唸**，結合辨識信心算出 0–100 分，並產生針對性的中文建議
（含 th、v/w、母音長短等常見錯誤提示）。全部在瀏覽器端完成，不上傳任何語音。

## 🚀 本地執行

因使用 ES Modules，需透過 HTTP 伺服器開啟（不可直接 file://）：

```bash
npx serve .      # 或 python -m http.server 8080
```

然後用 Chrome 開啟 http://localhost:8080。

## 🔒 隱私

語音辨識由瀏覽器原生 Web Speech API 處理，本站不蒐集、不上傳任何錄音或文字。
